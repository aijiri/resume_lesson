Foundations

What is HTML & CSS?

Before you write a single line of code, let's understand the two tools you'll use — and why they work together.

🧱

HTML — the structure

HTML is like the frame of a building. It defines what's on the page: your name, headings, paragraphs, and lists.

🎨

CSS — the style

CSS is the interior design. It controls colors, fonts, spacing, and layout — how everything looks.

Key concept

A resume webpage is just a text file saved as .html. Any browser (Chrome, Safari, Firefox) can open it and display it beautifully.

Key vocabulary you'll use today:

<tag> element attribute selector property value <head> <body>
💡 Tip: HTML uses tags to mark content. Almost every tag has an opening tag like <p> and a closing tag like </p>. Everything in between is the content.

On the right, you'll find a live code editor. In later steps, you'll type code there and click Run to see your resume take shape!

HTML

Step 1 — Build the skeleton

Every HTML page starts with the same basic skeleton. Think of it as the foundation before you add any rooms.

The boilerplate

Every webpage begins with <!DOCTYPE html> to tell the browser: "this is HTML." Then comes <html>, which wraps everything. Inside, <head> holds settings (like the title in the browser tab), and <body> holds what visitors actually see.

1
Look at the starter code in the editor. Notice how the tags are nested inside each other like boxes.
2
Find <title> inside <head>. Change the text to your own name, like: <title>Yuki Tanaka — Resume</title>
3
Inside <body>, add a heading tag: <h1>Your Name</h1> with your actual name.
4
Click Run ▶ to see the result in the Preview tab on the right.
reference — HTML skeleton
<!-- Every HTML file starts here -->
<!DOCTYPE html>
<html lang="en">

  <head>
    <meta charset="UTF-8">
    <title>Your Name — Resume</title>
  </head>

  <body>
    <h1>Your Name</h1>
    <!-- Content goes here -->
  </body>

</html>

Step 2 — Add your resume sections

Now let's fill your page with real content: contact info, education, experience, and skills.

Semantic HTML

Good HTML tells the browser what something is, not just what it looks like. Use <section> to group parts of your resume. Use <h2> for section headings, <p> for paragraphs, and <ul> with <li> for bullet lists.

1
After your <h1>, add a contact line using <p>: your email, city, and a link to your GitHub or LinkedIn if you have one.
2
Add an Education <section> with an <h2> heading. Inside, use <p> for your school name, degree, and graduation year.
3
Add a Skills <section>. Use a <ul> list where each <li> is one skill. Aim for at least 4 skills.
4
(Optional) Add an Experience section the same way as Education.
reference — section structure
<section>
  <h2>Education</h2>
  <p>Kyoto University</p>
  <p>B.A. Global Studies, 2026</p>
</section>

<section>
  <h2>Skills</h2>
  <ul>
    <li>HTML & CSS</li>
    <li>Microsoft Excel</li>
    <li>Japanese (Native)</li>
    <li>English (Advanced)</li>
  </ul>
</section>
💡 Links in HTML: Use <a href="..."> to make a clickable link. Example: <a href="mailto:you@email.com">you@email.com</a>

CSS

Step 3 — Style your resume

Right now your resume looks plain. CSS transforms it into something polished and readable. Switch to the CSS tab in the editor!

How CSS works

CSS rules follow this pattern: selector { property: value; }. The selector picks which HTML element to style. The property says what to change. The value says how to change it.

1
Switch to the CSS tab in the editor. Add a body rule to set the font, max-width, and background color.
2
Style h1 to control your name's font size and color. Try something like color: #185FA5;.
3
Style h2 to make section headings stand out — try adding a border-bottom like the example below.
4
Add spacing using margin and padding. Breathable spacing makes resumes much easier to read.
reference — CSS starter rules
body {
  font-family: Arial, sans-serif;
  max-width: 720px;
  margin: 40px auto;
  padding: 0 24px;
  color: #222;
  background: #fff;
}

h1 {
  font-size: 2rem;
  color: #185FA5;
  margin-bottom: 4px;
}

h2 {
  font-size: 1.1rem;
  text-transform: uppercase;
  letter-spacing: 0.08em;
  border-bottom: 2px solid #185FA5;
  padding-bottom: 4px;
  margin-top: 1.5rem;
}

section {
  margin-bottom: 1.5rem;
}

ul {
  padding-left: 1.25rem;
}
💡 Colors in CSS: You can use color names (blue), hex codes (#185FA5), or RGB values. Hex codes give you the most control — try coolors.co to find colors you like!

Challenge

Step 4 — Make it yours

You've built a working resume page! Now customize it to reflect your personality and go further with these challenges.

🎉 You've covered the essentials. Everything below is an extension — try as many as you like!
A
Change the color scheme. Pick 2 colors that feel like "you." Update your h1 color and h2 border to match.
B
Add a Google Font. Go to fonts.google.com, pick a font, copy the <link> tag into your <head>, and apply it to the body with font-family.
C
Add a two-column layout. Use CSS display: grid; grid-template-columns: 1fr 2fr; to put your name/contact on the left and sections on the right.
D
Add a hover effect. Make your links change color on hover: a:hover { color: #0F6E56; }
E
Save and share. Save your HTML file. Open it in your browser directly. Share the file with a classmate to review each other's resumes!
